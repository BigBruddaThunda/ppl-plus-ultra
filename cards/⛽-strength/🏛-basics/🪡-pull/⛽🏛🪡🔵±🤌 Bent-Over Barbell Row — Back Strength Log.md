# 🪡 Bent-Over Barbell Row — Back Strength Log 🪡

Barbell strength pull, prescribed and tracked | Lats, erectors, traps, biceps | 65–75 min

CODE: ⛽🏛🪡🔵

## 1. ♨️ WARM-UP

🤌 facio — move the tissue before you load it

- 30 sec Dead Hang (passive, breathe)
- 10 Scapular Pull-Up (depress then retract — two distinct movements)
- 10 Band Pull-Apart (shoulder-width, arms straight)
- 8/side Prone Y-Raise (floor, thumbs up)
- 8 Cat-Cow (thoracic emphasis)

═══════════════════════════════════════

## 2. ▶️ PRIMER

🤌 facio — ramp the hinge before loading it

- 5 🪡 Romanian Deadlift at 40% (hinge pattern, bar drags the shins)
- 3 🪡 Deadlift at 50% (setup from scratch every rep)
- 3 🪡 Deadlift at 60% (lats locked, breath held, move with purpose)

Rest: 90 sec between primer sets.

═══════════════════════════════════════

## 3. 🧈 BREAD & BUTTER

**A. Bent-Over Barbell Row (Overhand)**

🤌 facio — execute the prescription

Rest: 4 min between deadlift sets. 3 min between row sets.

**A. 🪡 Deadlift (Conventional)**

- Set 1: 80% × 5 (position before tension before pull)
- Set 2: 82.5% × 5 (hold the back angle through all 5)
- Set 3: 82.5% × 4 (rep 4 tells you if the load is right)
- Set 4: 77.5% × 5 (back-off — touch-and-go is fine here if the hinge stays sharp)

**B. 🪡 Bent-Over Barbell Row (Overhand)**

- Set 1: 77.5% × 5 (torso at or below parallel — if it rises, load is too heavy)
- Set 2: 80% × 5 (row to the hip, elbows drive back not up)
- Set 3: 80% × 5 (hold 1 sec at the top on every rep)
- Set 4: 75% × 6 (back-off, slightly higher rep — keep the torso locked)

## 4. 🧩 SUPPLEMENTAL

🤌 facio — secondary work, different angle

Rest: 2–3 min

**🪡 Barbell Shrug**

- Set 1: 82.5% × 5 (vertical pull, no rolling the shoulders)
- Set 2: 85% × 5 (pause at the top — half a second)
- Set 3: 82.5% × 6 (back-off — still heavy, slightly more volume)
