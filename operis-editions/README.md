# PPL± Operis Editions

Each file in this directory is a single edition of the PPL± Operis.

Path convention: `operis-editions/YYYY/MM/YYYY-MM-DD.md`

Each edition carries frontmatter with rotation engine data and featured zip codes.
Editions are the editorial record and the generation trigger for zip code workout population.

See `seeds/operis-architecture.md` for the complete Operis specification.
