# How the Abacus Works

An abacus is a training program inside Ppl±. There are 35 of them. Each one is a complete way to train — a set of 48 workouts that rotate across your week.

## The 35 Programs

Every abacus is a combination of an Order (how heavy) and a Type (which muscles). That gives you 7 × 5 = 35 distinct training archetypes.

Here's what that looks like:

| | Push | Pull | Legs | Plus | Ultra |
|---|---|---|---|---|---|
| **Foundation** | Push coaching | Pull coaching | Leg coaching | Power basics | Cardio basics |
| **Strength** | Heavy pressing | Heavy pulling | Heavy squats | Heavy power | Strength conditioning |
| **Hypertrophy** | Chest/shoulder growth | Back/bicep growth | Quad/ham growth | Core volume | Cardio volume |
| **Performance** | Press testing | Pull testing | Squat testing | Power testing | Conditioning testing |
| **Full Body** | Push-integrated flows | Pull-integrated flows | Leg-integrated flows | Full power flows | Movement flows |
| **Balance** | Tricep/delt correction | Bicep/grip correction | Calf/adductor correction | Rotational stability | Aerobic base |
| **Restoration** | Push recovery | Pull recovery | Leg recovery | Mobility/breath | Active recovery |

Each cell is an abacus. Select the ones that match how you want to train.

## What's Inside Each Abacus

Each abacus contains 48 workouts — 6 Axes × 8 Colors. The Axis changes the exercise character (classic lifts, athletic movements, isolation work, hardest variations, time-based protocols, or partner exercises). The Color changes the equipment and format (bodyweight, structured sets, high intensity, circuits, etc.).

That means within a single abacus, you get 48 different sessions that all share the same loading philosophy and muscle focus but vary in style, equipment, and structure.

## The Rotation

You don't pick a workout each day. The system picks it for you based on three gears:

**Gear 1 — Order by Weekday.** Each day of the week has an Order. Monday might be Strength. Tuesday might be Hypertrophy. The 7 Orders map to the 7 days.

**Gear 2 — Type by Rolling Calendar.** The 5 Types (Push, Pull, Legs, Plus, Ultra) rotate on a 5-day cycle starting January 1st. Since 5 and 7 are coprime numbers, the same Order-Type combination doesn't repeat for 35 days.

**Gear 3 — Axis by Month.** Each month has a dominant Axis that biases exercise selection. January might favor Basics. February might favor Functional. This creates seasonal variety.

The result: your daily workout is automatically determined. You open the app, and today's room is waiting.

## Combining Programs

You can run more than one abacus at a time. A common setup might be:

- **Strength Pull** + **Hypertrophy Push** + **Restoration Legs** — heavy back work, growth-focused chest work, recovery for legs.
- **Foundation Full Body** (everything) — if you want one program that covers all five Types with coaching-level loads.
- **Performance Push** + **Performance Pull** + **Performance Legs** — testing phase across all major lifts.

The rotation engine handles the scheduling. You select your programs, and the system weaves them into your weekly calendar.

## 48 Flavors

Because each abacus has 48 workouts (6 Axes × 8 Colors), you won't repeat a session for a long time even within a single program. The variety is structural — built into the math of the system, not randomly generated.

A "Strength Pull" workout on a Basics day with Structured format is a different room than a "Strength Pull" workout on a Challenge day with Intense format. Same muscles, same loading philosophy, completely different exercises, structure, and feel.
