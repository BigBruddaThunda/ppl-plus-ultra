## Relationship to Card Generation

When generating cards for a deck, the cosmogram (if populated) is
read after `CLAUDE.md` and `scl-directory.md` for deep context. If no
cosmogram exists, generation proceeds using `scl-directory.md` and
deck identity alone. Cosmograms are supplemental, not blocking.
