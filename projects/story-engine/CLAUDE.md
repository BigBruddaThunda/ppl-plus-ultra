## What This Project Is (Working Definition)

Narrative and video game architecture using SCL addressing. The hypothesis:
story structure, character arcs, world-building, and game mechanics can be
addressed using the same zip code grammar that organizes workouts and buildings.

Acts, scenes, characters, factions, locations, mechanics — all as zip-coded
content with Order (pacing phase), Axis (narrative character), Type (content domain),
and Color (format and tone).

This is a hypothesis, not a specification. It will be validated or revised during
intake processing and Ralph Loop sorting.
