## What This Project Is (Working Definition)

Real estate, urban planning, and civic design using SCL addressing. The hypothesis:
parcels, zones, districts, and civic systems can be addressed using the same zip code
grammar that organizes workouts and buildings.

Land use, building typology, program, infrastructure, governance — all as zip-coded
content with Order (development phase), Axis (design character), Type (content domain),
and Color (format and access tier).

This is a hypothesis, not a specification. It will be validated or revised during
intake processing and Ralph Loop sorting.
