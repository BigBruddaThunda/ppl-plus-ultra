  <implementation>
Port zip_converter.py logic to TypeScript. Use [...emojiZip] spread for emoji splitting (NOT split('')). Import all const objects and reverse indexes from canvas/src/types/scl.ts. Use CLAUDE.md canonical operator emojis in OPERATOR_TABLE, not the stale Python values.

Functions to implement:
1. emojiToZip(emojiZip: string): string — splits emoji string, looks up each position via reverse indexes, concatenates digits
2. zipToEmoji(numericZip: string): string — validates range, looks up each digit in const objects, concatenates emojis
3. isValidZip(zip: string): boolean — returns true if 4 digits, each in valid range (order 1-7, axis 1-6, type 1-5, color 1-8)
4. zipToDeck(zip: string): number — accepts emoji or numeric, returns (order - 1) * 6 + axis
5. deriveOperator(emojiZip: string): { emoji: string; name: string } — reads axis (position 2) and color (position 4), checks polarity, returns operator

Internal helper:
- validateNumericZip(numericZip: string): void — throws on invalid format or out-of-range digits
  </implementation>
