**computeRawVector() in index.ts:**
- Import only ORDERS_WEIGHTS, AXES_WEIGHTS, TYPES_WEIGHTS, COLORS_WEIGHTS (the 4 dials)
- Do NOT import BLOCKS_WEIGHTS or OPERATORS_WEIGHTS in computeRawVector
- Re-export all 6 tables and the types for downstream consumers
- Algorithm:
  1. Allocate Float32Array(WEIGHT_VECTOR_LENGTH + 1) — 62 slots, slot 0 unused
  2. Step 1: Set primary +8 at each dial's W position (orderPos maps directly, axisPos+7, typePos+13, colorPos+18)
  3. Steps 2-3: Iterate affinities and suppressions for each of the 4 active dial entries, summing into the vector
  4. Clamp all positions to [-8, +8] with Math.max(-8, Math.min(8, v))
  5. Return the Float32Array
