Port the exercise library to a typed TypeScript interface with fuzzy-matchable alias layer.

Purpose: The exercise dictionary is the authoritative lookup table for exercise name detection (PARS-03), equipment mention detection (PARS-04), and body part detection (PARS-05). It bridges natural language exercise references to structured SCL metadata. The alias layer enables the Phase 3 fuzzy matcher to resolve gym slang and abbreviations.

Output: ExerciseEntry TypeScript interface, exercises.json with ~2,085 entries including aliases, and tests.
</objective>
