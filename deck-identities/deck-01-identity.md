# Deck 01 Identity — 🐂🏛 Foundation Basics

## 1) Deck Philosophy
🐂 Foundation (≤65% load, 8–15 reps, 60–90s rest, max difficulty 2/5, low CNS demand) meets 🏛 Basics (bilateral, stable, barbell-first, time-tested fundamentals). This deck is the system ground floor: bench, squat, deadlift, row, press, curl, and extension patterns taught at loads where position quality is the metric. It is not beginner-only; experienced lifters return here whenever a pattern is relearned. The ceiling is absolute: nothing exceeds 65%, nothing drops below 8 reps, and nothing exceeds difficulty 2/5.

## 3) Color Differentiation Logic
- ⚫ Teaching — coaching-dense pattern work with clear verbal setup and bar path cues.
- 🟢 Bodyweight — no-equipment classic patterns (push-ups, rows, squats, crawls, jumps).
- 🔵 Structured — prescribed sets/reps/rest with repeatable tracking across weeks.
- 🟣 Technical — precision drill lens on classic lifts using pause, tempo, and position holds.
- 🔴 Intense — higher-volume foundational pattern work under moderate fatigue; no GOLD work in this deck because 🐂 difficulty ceiling is 2/5.
- 🟠 Circuit — station rotation through basics with loop logic and no barbells.
- 🟡 Fun — controlled exploration with classic movements and varied but valid exercise picks.
- ⚪ Mindful — slow-tempo foundational work with breath-paced execution and longer settling rest.
