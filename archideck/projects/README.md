# Projects

Each project in the Archideck has its own directory with at minimum:
- `CLAUDE.md` — Project-specific agent operating instructions
- `whiteboard.md` — Current state, active task, next action

Projects use the SCL kernel as shared language. Domain-specific vocabulary extends SCL, never replaces it.
